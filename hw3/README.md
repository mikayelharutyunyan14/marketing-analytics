# Telco Customer Churn — Survival Analysis & CLV

DS 223 - Marketing Analytics

Homework 3

We use parametric survival models to understand what drives customer churn in a telecom dataset, then estimate each customer's lifetime value based on their predicted survival trajectory.

## Files

- `telco_survival_clv.ipynb` — full analysis, code, and report
- `telco.csv` — dataset (telecom subscribers)
- `requirements.txt` — Python dependencies

## How to Run

```bash
pip install -r requirements.txt
```
