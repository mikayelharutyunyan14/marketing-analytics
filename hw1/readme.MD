This repository contains supporting materials for the analysis of solar PV adoption, focusing on global trends and the Netherlands residential market. The analysis uses the Bass diffusion model to estimate adoption parameters and forecast future growth.

Key points:  
- Data covers global installed solar PV capacity (2000–2024) and Dutch residential Solar PV systems (2019–2024).  
- Bass model parameters were estimated using nonlinear least squares (nls) and the diffusion package.  
- Adoption is primarily driven by imitation rather than early innovators.  
- Forecasts suggest continued growth through 2035, with the market approaching—but not yet reaching—saturation.  

For full details, graphs, and discussion, see the accompanying report.pdf and analysis.pdf files.
